### Información del Ejercicio Técnico

**Lenguaje utilizado**: JavaScript

**Versión del lenguaje**: Node.js 18.x

**Instrucciones de Ejecución**:
1. Abre el sandbox: https://codesandbox.io/p/sandbox/y5n2kn.
2. Haz clic en "Run" para ejecutar el archivo `index.js`.

**Notas adicionales**:
- El archivo contiene dos funciones:
  - `problem(nums)`: Resuelve el problema con un enfoque iterativo.
  - `problemOptimized(nums)`: Optimiza la solución con un enfoque de paso único.
